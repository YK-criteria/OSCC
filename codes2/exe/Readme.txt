- feature2bat.bat
Extract Features
[Usage]
 in feature2bat.ini, specify these.
 [exe] filename of executable
 [folder] Name of folder containing image files from which features will be extracted
 [output] Output folder name

 Execute "FeatureBat.exe featurebat2.ini" to get featurebat2.bat, 
 then execute "featurebat2.bat".
